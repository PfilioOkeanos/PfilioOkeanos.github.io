What it does

Pfilio's Auto-Target Lightning creates animated target-seeking lightning effects that can react to a target object or target collection. It is designed for stylized VFX, sci-fi scenes, magic attacks, energy effects, motion graphics, cinematic shots and experimental Blender animations.

The included shader setup can create bright electric bolts, darker lightning looks, glowing energy, liquid-like spline effects and more experimental VFX results.

Main controls

  • Lightning intensity and glow intensity
  • Thickness, scale and branch count
  • Animated jitter / noise movement
  • Target distance and target influence
  • Start / end bias and direction controls
  • World light direction and shader color variation

Important setup note

Create and assign materials to the Source and Target objects first. After that, add the lightning shader for the node setup.

This order is important so the Source and Target objects do not accidentally use the node shader.
